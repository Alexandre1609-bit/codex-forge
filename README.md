# CODEX FORGE | BUT INFORMATIQUE

> **Objectif :** Centraliser l'apprentissage et la montée en compétences alignée sur le programme du **BUT Informatique** (Référentiel National).
> 
## Architecture du Monorepo

Ce dépôt est structuré pour durer 3 ans, de l'initiation à l'expertise Cloud/DevSecOps.

```text
codex-forge/
│
├── ANNEE_1_FONDATIONS/       # BUT 1 : Bases solides (Algo, Système, SQL)
│   ├── R1.01_Initiation_Dev/    # Python & Java
│   ├── R1.04_Systeme/           # Scripting Bash & Linux
│   └── R1.05_Bases_de_Donnees/  # SQL & Intégration Python (Zone active)
│
├── ANNEE_2_APPROFONDISSEMENT/ # BUT 2 : Appli Web, Réseaux, Sécurité
└── ANNEE_3_EXPERTISE/        # BUT 3 : DevSecOps, Kubernetes, Cloud
```
